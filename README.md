# learn-django

### description

```
simple shopping web with django and custom user models
```


### Tools

- python 3.7
- django 2.2
- pillow 8.3
- django-environ 0.7
- psycopg2 2.9
- postgres

### Env variable

- DB_ENGINE=
- DB_NAME=
- DB_USERNAME=
- DB_PASSWORD=
- DB_HOST=
- DB_PORT=
- SECRET_KEY=

```
- setup all tools
- run: python3 manage.py runserver
```
